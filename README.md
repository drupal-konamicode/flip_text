Flip Text jQuery Plugin for the Drupal Konami Code module
=========

jQuery plugin 

https://www.paulirish.com/2009/flip-text-upside-down-jquery-plugin/

Installation
------------
Install with composer:

> composer require drupal-konamicode/flip_text

------------
